Dynamic Causal Network Modeling

Conceptual Framework and Project Information

DYNOTEARS Concept Overview

Causal structure learning from time-series data utilizes Dynamic Bayesian Networks (DBNs) to map connections across temporal states. The DYNOTEARS (Dynamic NOTEARS) algorithm extends structural learning by formulating DBN parameters into a continuous optimization problem with algebraic acyclicity constraints.

Key Visual Mechanisms:

1. Lagged Connections (Past-to-Present): Directed edges that run from _lag1 to _lag0 nodes. These show how historical fluctuations in one node influence the present value of another, confirming traditional understandings of causality where past events trigger present consequences.

2. Intra-Slice Connections (Present-to-Present): Causal paths between nodes in the same time frame (_lag0 to _lag0). An acyclicity constraint prevents loops, reflecting instant contagion paths without retroactively affecting the past.

3. Thresholding: Causal learning algorithms generate dense networks including weak, noisy relationships. Setting a threshold filter allows analysts to isolate the core systemic structure and trace propagation pathways clearly.

Causal Modeling in Financial Risk

Systemic risk analysis relies on identifying structural networks between institutions or assets. Traditional measures (like correlation matrices) fail to establish the direction of impact. Applying Dynamic Bayesian Networks helps trace trigger nodes and identify propagation cascades across systemic nodes.

Simulation Design

This interactive frontend replicates causal structures calculated by learning algorithms. It shows how adjustment of the weight threshold filters noise, leaving only key propagation paths (like trigger asset connections).
